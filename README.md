## Instalation
First of all, must have installed in your machine NodeJS.

1. Clone the repo:
   ```sh
   git clone https://github.com/Silviomjr/Moveit
   ```
2. Install locally:
   ```sh
   yarn
3. To run in your localhost:
    ```sh
    yarn dev
    ```
